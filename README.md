# antiplatka

To install dependencies:

```bash
bun i
```

```bash
bunx prisma db push
```

To run:

```bash
bun run build
```

```bash
bun run start
```

Or:

```bash
bun run dev
```

This project was created using `bun init` in bun v1.2.2. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
